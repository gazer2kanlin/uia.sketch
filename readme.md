此工具程式利用虛擬網格線條，輔助業餘素描愛好者(例如我自己)，在構圖時增加準確性。

* 提供雙網格套疊

* 可自訂網格顏色

* 可調整網格間距與角度

* 可利用鍵盤調整照片大小

* 可利用鍵盤調整網格間距與角度
