此工具可幫助業餘繪圖愛好者(例如我自己)，利用虛擬格線的套疊，增加構圖時的準確性。

目前有的格線：

* 正方形
* 圓形
* 正三角形

## 功能

* 多重套疊

* 可自定義格線顏色

* 可調整格線間距與角度

* 可利用鍵盤調整格線間距與角度

* 可利用鍵盤調整照片大小
